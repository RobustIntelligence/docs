[3]:

rime_client = Client(CLUSTER_URL, API_TOKEN)

Python SDK package and server are on different versions. The Python SDK package is on version 2.0.0rc10.post0+git.22a34719.dirty, while the server is on version 2.0.0-rc.8. In order to make them be on the same version, please install the correct version of the Python SDK package with `pip install rime_sdk==2.0.0-rc.8`

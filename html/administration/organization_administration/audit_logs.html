Audit LogsÔÉÅ

  • Admins can access a records of events and activity on the RI platform for the last 12 months. Eg: Projects created, API token generated, User Deleted

  • Audit logs can be exported as a CSV file from the General Settings page under Organization settings.

  • Audit logs consist of the event description, time of the event, API token name, username, and user role.

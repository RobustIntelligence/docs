CLI

These walkthroughs illustrate how to test your ML models using the different RIME modalities. Each walkthrough provides steps to run the examples using bundled sample data or your own models and data.

Ad-hoc model testing can be performed with the RIME Python Package, which exposes both a command-line interface and python modules.

Both ad-hoc model testing and programmatic interactions can be performed with the RIME SDK, which exposes a REST client that can be integrated in your existing ML pipelines.

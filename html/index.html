Welcome to RIME!

The Robust Intelligence Model Engine (RIME) enables comprehensive testing of data and model performance at the pre-development and post-development stages of the model life cycle.

RIME documentation covers several broad topic areas. If you’re new to RIME or want to familiarize yourself with core concepts of machine learning, RIME at a Glance provides an overview.

The most recent version of this documentation is always available here.

To find information on how to achieve day-to-day data science tasks with RIME, look in the Testing and monitoring models with RIME section, which discusses actively using RIME to evaluate datasets and models. Configuring alerts, setting up integrations, and running tests on uploaded models are all discussed here.

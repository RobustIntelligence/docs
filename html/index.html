Welcome to RIME!

The Robust Intelligence Model Engine (RIME) enables comprehensive testing of data and model performance at the pre-development and post-development stages of the model life cycle. Users already familiar with RIME who want to know what’s new in the most recent release can find details in the Release Notes.

  • Release Notes

RIME documentation covers several broad topic areas. If you’re new to RIME or want to familiarize yourself with core concepts of machine learning, RIME at a Glance provides an overview, as well as a comprehensive walkthrough of a RIME local trial.

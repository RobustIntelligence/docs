Managing Model Governance
================================

Robust Intelligence provides model governance capabilities alongside the stress and continuous test suite.

.. toctree::
   :maxdepth: 1

   model_governance/model_cards.md

Users concerned with model risk management and compliance need standardized and automated model documentation. The Model Card Builder allows users to automate and customize model documentation to meet internal and regulatory reporting requirements.

.. toctree::
   :maxdepth: 1

   model_governance/workspace_overview.md

Users in compliance and executive roles may need to view metrics and model health for all projects in a workspace. The Workspace Overview serves as a dashboard for all models in production within a workspace, providing model health status and the ability to track models to any custom metric.

.. toctree::
   :maxdepth: 1

   model_governance/project_overview.md

Executives, managers, or data scientists might need to view a detailed risk snapshot for a project. The Project Overview is the landing page of a project in Robust Intelligence. It proves a summary for all testing failure events and information about the production model.

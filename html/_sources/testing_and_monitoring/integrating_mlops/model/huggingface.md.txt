# Hugging Face

Robust Intelligence now supports an API that downloads Hugging Face models by providing the
Hugging Face Model's URI. To use this functionality, specify a HuggingFace [model configuration](../../preparing_your_models_and_datasets/model_source.md).

### Hugging Face Classification Model

```python
{
    model_info = {
        "hugging_face" :{
            "model_uri": "cross-encoder/nli-MiniLM2-L6-H768",
        }
    }
}
```

# s3

Robust Intelligence supports loading data directly from s3.

Follow all of the instructions in the generic [Data Configuration](../../preparing_your_models_and_datasets/data_source.md) guide, except the path to the data should no longer be a local path but instead an s3 storage path

# Configuration example

```python
{
    "connection_info": {"data_file": {"path": "s3://path/file.csv"}},
    "data_params": {
        "label_col": "bar",
        "sample": False,
        "text_features": ["text_1", "text_2"],
    },
}
```

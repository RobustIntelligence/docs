# Model Configuration

Configuring a model source can be done by specifying a mapping in the RIME JSON
configuration file, under the `model_info` argument. [Specify a Model](specify_model.md)
discusses how to create a Python model file.

## Template
```python
{
    "model_info": {
        "model_path": "$PATH";
    }
    ...
}
```

## Arguments

| Parameter | Type | Description |
|-----------|------|-------------|
| model_path | ModelPathInfo | Path to the Python model file.  |
| &nbsp;&nbsp;&nbsp;&nbsp; path | String | Contains the actual path. |
| hugging_face | HuggingFaceModelInfo | The URI to a HuggingFace model. |
| &nbsp;&nbsp;&nbsp;&nbsp; model_uri | String | Contains the actual URI. |
| &nbsp;&nbsp;&nbsp;&nbsp; kwargs | String | Arguments to pass to HuggingFace along with the URI. |

Specify exactly one of `model_path` or `hugging_face`.

# Hugging Face Datasets

RIME now offers a data provider that natively integrates with Hugging Face's [`load_dataset`](https://huggingface.co/docs/datasets/v2.3.2/en/package_reference/loading_methods#datasets.load_dataset) API. To use this functionality, specify a HuggingFace [data configuration](../../../reference/data_source.md).

### Data Info Configuration Example

```python
{
    "connection_info": {
        "hugging_face": {
            "dataset_uri": "sst2",
            "split_name": split_name,
            "loading_params_json": json.dumps({}),
        }
    },
    "data_params": {
        "label_col": "label",
        "text_features": ["sentence"],
        "sample": True,
        "nrows": 10
        "class_names": ["Nimitz", "Arkleigh", "Zumwalt"],
    },
}
```

Model Configuration
===================

Configuring a model source can be down by specifying a mapping in the RIME JSON
configuration file, under the `model_info` argument.


### Template
```python
{
    "model_info": {
        "path": "/path/to/model.py"        (REQUIRED)
    }
    ...
}
```

### Arguments

- **`path`**: string, ***required***

    Path to PYTHON model file. For instructions on how to create this Python
    model file, please see [Specify a Model](../../walkthroughs/cli/specify_model.md).
